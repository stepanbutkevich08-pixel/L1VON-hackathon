# Оптимальное размещение клиник

## Описание проекта
Проект решает задачу оптимального размещения медицинских клиник в городе на основе данных о клиентах, инфраструктуре и транспортной доступности.

Цель — **минимизировать среднее время в пути клиентов до ближайшей клиники** при строгом ограничении на максимальную загрузку каждой клиники, а также проанализировать качество размещения с точки зрения социальной и инфраструктурной доступности.

Решение реализовано в виде одного Python-скрипта без внешних конфигурационных файлов. Все параметры задачи зафиксированы непосредственно в коде, что обеспечивает воспроизводимость результатов.

---

## Входные данные

Ожидается файл `Data.csv` со следующими колонками:

| Колонка | Описание |
|-------|----------|
| `x`, `y` | Координаты клиента |
| `client_age` | Возраст клиента |
| `clinick_distance` | Тип транспортной доступности (1 — метро, 2 — трамвай, 3 — автобус, 4 — такси) |
| `density_area` | Плотность населения района |
| `park_distance` | Расстояние до ближайшего парка |
| `vulnerable_group_density` | Плотность уязвимых групп |
| `social_infrastructure_rating` | Оценка развитости социальной инфраструктуры |
